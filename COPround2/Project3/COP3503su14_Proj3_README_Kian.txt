Project 3 RecipeBook and Inventory List Program

What this program does:

	This program reads, names, stores, and operates on recipes. Operations include combining a set 
	of recipes to arrive at a combined ingredient list, combined equipment, and combined instructions. 
	They also include taking inventory lists of available ingredients and equipment and subtracting them
	from combined lists to arrive at shopping lists of needed items.
	
To install:
	1. Navigate to the directory where the program is located through command prompt (hint: 'cd')
	2. run the Makefile by typing ¬Åmake on the command prompt
	3. To run the code, type './a.out' followed by any commands to the program.
